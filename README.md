# Rulesmith (法则锻造台)
> 专为 TRPG 跑团桌面桌游（如《显微镜 Microscope》）设计的强同步在线“调色板”协作工具。

Rulesmith 是一款轻量级的全栈 Web 应用，旨在游戏开局前帮助所有玩家精确商定游戏世界的框架界限（**必定包含的元素（是/YES）** 和 **被绝对禁止的元素（否/NO）**）。

## ✨ 核心特性
- **双极阵营展现**：极简清爽的深/浅色界面设计，一目了然世界历史的“Yes”和“No”。
- **绝对公平锁 (+2 规则验证)**：强力引擎拦截：当任何玩家提出的条目数量超出全场其他人最低数量的两条及以上时，其提交按钮将被强制禁用，直到别的玩家追上进度。该机制在前端UI与后端Socket双重保障，绝不允许任何开挂。
- **阶段共识流转**：当所有玩家点击“我已完成”时，工作区将智能退位，弹下“共识阶段”视窗，供讨论使用。
- **纯粹、无痕、轻量化**：告别一切繁重的数据库。使用了纯内存级 WebSocket 字典树：只要全房人下线即可自动启动 Garbage Collection，来无影去无踪。

## 🛠 本地开发与联机测试

只需一步，即可在 Windows 下体验“双手互博”的多开测试：

1. 双击运行工程根目录下的 `start-local.bat` 脚本
2. 等待两个防丢包后台窗口拉起后，用浏览器（加上几个无痕窗口）在本地访问 `http://localhost:5173/`。
3. 输入相同的房间代码即可一起编织世界观！

## 🐳 Linux VPS 傻瓜式一键部署 (Docker)

本工程特别为了方便自建站服务器而开发了「单一容器多阶段发版」架构。无需配置 Nginx 等繁杂环境，自动接管全分发职责。

在您的 Linux 服务器上打开带有本源码的文件夹：

```bash
# 赋予执行权限并直接启动！
chmod +x deploy.sh
./deploy.sh
```

此后，只需将 IP 发给您的玩家即可开始创世之旅。
